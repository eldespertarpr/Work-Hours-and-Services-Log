WORK HOURS AND SERVICES LOG - MINI APP

Included:
- English interface.
- Tabs: Info, Entries, Notes, Summary, Print View.
- Work day entries with:
  - Date
  - Start Time
  - End Time
  - Ends next day
  - Break Minutes
  - Calculated Hours
  - Work Performed
- Automatic total hours.
- Print layout similar to a formal Work Hours and Services Log.
- Letter landscape printing.
- Save data locally on device/browser.
- Ready for GitHub Pages or Netlify.

How to use:
1. Open index.html in a browser.
2. Fill out Info.
3. Add work day entries.
4. Add notes if needed.
5. Go to Summary or Print View.
6. Use Print / Save PDF.

Phone installation:
- Upload all files to GitHub Pages or Netlify.
- Open the site on your phone.
- Use Add to Home Screen.
